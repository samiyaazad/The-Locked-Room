# The Locked Room

**By:** Samiya Azad

---

## About the Game
*The Locked Room* is a choice-based visual novel made with Ren’Py.  
Wake up in a strange room and make decisions to find different endings!  

---

## How to Play
1. Launch the game (Windows zip).  
2. Read the story and make choices.  
3. Discover the endings:
   - Good Ending 🎉 → escape safely  
   - Bad Ending ☠️ → get trapped  
   - Secret Ending 🗝️ → find hidden treasure  

---

## Features
- Multiple endings based on your choices  
- Secret paths and hidden surprises  
- Colorful backgrounds and character sprites  
- Easy-to-play interactive story  

---

## Requirements
- Windows 7 or later (or Mac/Linux if built)  
- No installation needed — unzip and run
